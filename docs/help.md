# Todo-list example
```
Todo-list example

- ToDo List features
   - Syntax highlight
      - This the basic list syntax. Each item starts with a dash followed by a space and the item description.
      - Sub-items can be made by indenting the item with tabs (not spaces)
      - You can indent/unindent multiple lines by selecting them and pressing Tab or Shift+Tab
      - Task items can be highlighted, like this:
         -- Item highlight 1
         --- Item highlight 2
         ---- Item highlight 3
         ----- Item highlight 4

      - Done Items
         - Basic syntax: Plus signs instead of dashes
         - Shortcut: Ctrl+/ (Ctrl+; on ABNT keyboards)
         + Done item Example
         ++ Done Item Example

   - List Folding
      - Fold arrows on the left, you can click them or press Ctrl+Q on the line to fold/unfold items.

   - Main shortcuts
      - Toggle done/undone: *Ctrl+/* (Ctrl+; on ABNT keyboards)
      - Code folding: *Ctrl+Q*
      - Move lines up/down: *Ctrl+Shift+Up, Ctrl+Shift+Down*
         - Try to switch this line
         - With this one
      - Delete line: *Ctrl+Shift+k*
```
